CRITICAL (2)

C-001

API key가 git 히스토리에 노출 (커밋 a3f2e1d, 2024-08). .env는 gitignore되었으나 초기 커밋에서 평문 노출 후 revert만 수행.

C-002

GitHub Actions workflow에 pull_request_target + checkout 조합 사용. 외부 PR에서 시크릿 탈취 가능.

HIGH (4)

H-001

SQL 쿼리에 f-string 보간 사용 (f"SELECT * FROM {table}"). Parameterized query 미적용.

H-002

H-003

LLM API 호출 결과를 검증 없이 eval()로 실행. 프롬프트 인젝션 → 코드 실행 가능.
