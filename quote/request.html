자주 묻는 질문

표준 제품은 24시간 내, 맞춤형 제품은 48시간 내에 정확한 견적을 발송해드립니다. 긴급한 경우 전화 연락 시 당일 견적도 가능합니다.

표준 제품은 1미터부터 주문 가능합니다. 맞춤형 제품의 경우 최소 주문량이 있을 수 있으므로 견적 요청 시 상담받으시기 바랍니다.

표준 제품의 경우 무료 샘플(최대 30cm)을 제공합니다. 맞춤형 제품의 샘플은 별도 비용이 발생할 수 있으며, 정식 주문 시 샘플 비용을 차감해드립니다.

FDA, USP Class VI, ISO 등 각종 인증서와 재료 성적서(COA), 시험성적서를 제공합니다. 특별한 인증서가 필요한 경우 견적 요청 시 명시해주세요.
